<Update label="v1.2.0">

<Tip>

  work perfectly. If you experience any issues, please disable it.

1. **Added "Skip NativeLanguage Translation" option**

   Users can now choose to skip translations for their native language, allowing for more precise and customized results.

   ![](/images/d33.png)

2. **Introduced OpenAI-Like Translation API**

   All translation APIs compatible with OpenAI's API can now be used under the OpenAI-Like framework, providing enhanced flexibility and improved translation accuracy.

   ![](/images/d34.png)

</Update>
