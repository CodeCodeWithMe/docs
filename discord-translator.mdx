## Quick Input Translation

You can quickly translate the input content in the input box

### Overview

<video controls>
  <source src="/videos/quick-input-translator.mp4" type="video/mp4" />
</video>

### How To Use

When you enter "Nice to see you /ko" in the input box, it will be translated into Korean

Input Content : Nice to see you

End Symbol: /

Language Code: ko

<Tip>
  "ko" is the language code for Korean. To translate to another language, simply
  replace "ko" with the corresponding language code.
</Tip>

## Auto Translation

Real-time translation of chat content

### Overview

<video controls>
  <source src="/videos/auto-translator.mp4" type="video/mp4" />
</video>

### How To Use

Open the auto translation switch option in the extension options page.
When you receive a message, it will be translated into the language you set in the options.
