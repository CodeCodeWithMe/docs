## How to Activate Premium Features

<Steps>
  <Step>
    Open the Extension Options Page

    ![](/assets/10.png)

  </Step>

  <Step>
    Select the License Tab and Activate Your License

    ![](/assets/11.png)

  </Step>

  <Step>
    Refresh the Web Page

    - After activating your license, refresh the web page for the changes to take effect.
    - If it still doesn’t work, try reopening Chrome.

  </Step>
</Steps>

<Callout>
  If your license still doesn’t activate after several attempts, please contact
  our support team at **support@codecode.me** for assistance.
</Callout>
