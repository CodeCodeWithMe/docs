## Prefer a simpler option?

If the manual installation feels too complicated, you can try **Discord Translator Lite**, developed by one of our team members.

This lightweight version supports **automatic translation only** and can be installed directly from the Chrome Web Store:

https://chromewebstore.google.com/detail/hghdelkfmfjhfjghfgefhlanfflmijjl?utm_source=item-share-cb

## Discord Translator v3.0.3 Installation Guide

Follow the steps below to install the latest version of the extension:

1. **Disable the old version and deactivate your license**

   Disable v3.0.2 and deactivate your license.

   If you’re not sure how, watch this tutorial:

   [How to disable a Chrome extension](https://youtu.be/-eku8HZ-1RI)

2. **Download the latest version**

   Download v3.0.3 from the link below:

   [Download Link](https://drive.google.com/file/d/1rx2QMj5QBtz6mp5-qUFKpBLH-BignCSs/view?usp=sharing)

3. **Unzip the file**

   Extract the downloaded ZIP file to a local folder.

4. **Install the extension**

   Follow this video tutorial to install the extension:

   [Installation Guide Video](https://youtu.be/oswjtLwCUqg)

5. **Activate the extension**

   Enter your license key to activate the extension.

6. **License issues**

   If you encounter any license-related problems, reset your license:

   [Reset License Tutorial](https://doc.codecode.me/troubleshooting/how-to-reset-license#creem)

7. **If the extension is not working properly**

   - Refresh your Discord page.
   - If that doesn’t work, close your browser completely and reopen it.
