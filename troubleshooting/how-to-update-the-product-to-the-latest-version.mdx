## Chrome Extension

There are two ways to update the product to the latest version.

### Reinstall the Extension

1. Uninstall the current extension from your browser.

2. Go to the [Chrome Web Store](https://chrome.google.com/webstore/) to install the latest version of the extension.

3. After installation, you may need to reconfigure the settings and reactivate your license.

### Manually Update the Extension

<Steps>
  <Step title="Open Chrome Extensions Page">
    <img src="/images/dt/5.png" width="50%" />
  </Step>
  <Step title="Enable Developer Mode">
    Enable the developer mode by clicking the toggle button at the top right
    corner.
    <img src="/images/dt/6.png" width="100%" />
  </Step>
  <Step title="Click Update Button">
    Click the "Update" button to update all extensions to the latest version.
    <img src="/images/dt/7.png" width="100%" />
  </Step>
  <Step title="Check the Version">
    After updating, you can check the version of the extension to ensure it has
    been updated to the latest version.

    [How to Check the Product Version?](/troubleshooting/how-to-check-the-product-version)

  </Step>
  <Step title="Disable Developer Mode">
    You can disable the developer mode by clicking the toggle button again.
  </Step>
</Steps>
